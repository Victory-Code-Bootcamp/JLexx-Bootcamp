Lesson Challenge

I learned about the GitHub workflow process in terms of creating and deleting branches, how to rename branches and delete them locally and remotely, and about code reviews and merging a branch into the main branch.

Summary of Git Commands

  • To view a list of all branches:
    • git branch
  • To rename a branch (from old-name to new-name):
    • git branch -m old-name new-name
  • To delete a branch locally:
    • git branch -d branchname
  • To delete a branch forcibly (useful when the branch is not fully merged):
    • git branch -D branchname
  • To delete a branch on a remote repository:
    • git push origin --delete branchname
  • To create new branches, use:
    • git checkout -b branchname, or
      git branch branchname
